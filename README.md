# AgentDB

AgentDB is an AI-powered supply chain decision intelligence system built on Databricks.

## Features

- Demand forecasting
- Disruption detection
- Recommendation generation
- Agentic orchestration
- MLflow observability

## Architecture

Data Layer
→ Forecast Layer
→ Agent Layer
→ Recommendation Layer

## Tech Stack

- Databricks
- Delta Lake
- Unity Catalog
- Spark
- MLflow
- Python
